Chapitre 1 : Les Bases des Algorithmes

Un **algorithme** est une série d'instructions claires et précises permettant de résoudre un problème. Voici les concepts fondamentaux :

  • Variable : Un espace où stocker des données (par exemple, un nombre, un texte).
  • Instruction : Une action à exécuter (comme ajouter deux nombres).
Début
  Lire A, B
  S <- A + B
  Afficher S
Fin

Chapitre 2 : Les Structures de Contrôle

Les **structures de contrôle** permettent de décider quelles instructions exécuter ou de répéter des actions :

Début
  Lire X
  Si X > 0 Alors
    Afficher "Positif"
  Sinon
    Afficher "Négatif"
  FinSi
Fin

Chapitre 3 : Les Structures de Données

Les **structures de données** permettent de stocker plusieurs informations :

Début
  Tableau <- [2, 4, 6, 8, 10]
  Somme <- 0
  Pour Chaque Nombre dans Tableau Faire
    Somme <- Somme + Nombre
  FinPour
  Afficher Somme
Fin
