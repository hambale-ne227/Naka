Chapitre 1 : Les Bases des Algorithmes

Un **algorithme** est une série d'instructions claires et précises permettant de résoudre un problème. Voici les concepts fondamentaux :

  • Variable : Un espace où stocker des données (par exemple, un nombre, un texte).
  • Instruction : Une action à exécuter (comme ajouter deux nombres).
  • Structure logique : Le flux logique des étapes à suivre.

Exemple : Trouver la somme de deux nombres :

Début
  Lire A
  Lire B
  S <- A + B
  Afficher S
Fin

Chapitre 2 : Les Structures de Contrôle

Les **structures de contrôle** permettent de décider quelles instructions exécuter ou de répéter des actions.

  • Condition (Si ... Alors ... Sinon) : Permet de prendre une décision.
  • Boucles (Pour, Tant que) : Permettent de répéter une action plusieurs fois.

Exemple : Trouver si un nombre est positif :

Début
  Lire X
  Si X > 0 Alors
    Afficher "Positif"
  Sinon
    Afficher "Négatif"
  FinSi
Fin

Exemple de boucle : Afficher les nombres de 1 à 5 :

Début
  Pour I de 1 à 5 Faire
    Afficher I
  FinPour
Fin

Chapitre 3 : Les Structures de Données

Les **structures de données** permettent de stocker et d'organiser plusieurs informations en mémoire :

  • Tableaux : Une liste de données du même type.
  • Enregistrements : Une structure regroupant plusieurs données différentes.

Exemple de tableau : Ajouter 5 nombres :

Début
  Tableau <- [2, 4, 6, 8, 10]
  Somme <- 0
  Pour Chaque Nombre dans Tableau Faire
    Somme <- Somme + Nombre
  FinPour
  Afficher Somme
Fin
