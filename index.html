Chapitre 1 : Les Bases des Algorithmes

Un algorithme est une série d'instructions organisées pour résoudre un problème. Concepts clés :

  • Entrées : Données initiales fournies.
  • Instructions : Étapes pour résoudre le problème.
  • Sorties : Résultat final attendu.

Exemple : Trouver la somme de deux nombres.

Chapitre 2 : Les Structures de Contrôle

Les structures de contrôle permettent de gérer l'ordre d'exécution :

  • Conditionnelles : Permettent de choisir entre plusieurs actions (ex. : "Si" ou "Sinon").
  • Boucles : Permettent de répéter des instructions (ex. : "Pour", "Tant que").

Exemple : Afficher les nombres de 1 à 10.

Chapitre 3 : Les Structures de Données

Les structures de données sont des façons d'organiser et de stocker des informations :

  • Tableaux : Collections d'éléments de même type.
  • Listes : Suites d'éléments pouvant être modifiés dynamiquement.
  • Dictionnaires : Couples clé-valeur pour un accès rapide.

Exemple : Stocker et afficher les noms de 5 étudiants.
